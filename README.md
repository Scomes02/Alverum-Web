# 🌐 Alverum - Landing Web de Servicios de Seguros
### 🔗 Demo online: https://alverum-seguros.netlify.app

## 📄 Descripción

Proyecto de una página web orientada a la presentación de servicios de seguros, diseñada como una landing page moderna, responsive y optimizada para conversión.

El sitio permite a los usuarios conocer la empresa, visualizar las aseguradoras con las que trabaja y realizar consultas mediante un formulario de contacto funcional.

La aplicación se enfoca en una arquitectura simple y eficiente del lado del cliente, separando la estructura, estilos e interactividad para mantener un código limpio, mantenible y escalable.

Además, se integran servicios externos para la gestión de formularios y despliegue en la web, garantizando una solución rápida y efectiva sin necesidad de backend complejo.

## ✨ Características principales

Landing Page Profesional: Diseño moderno enfocado en la presentación clara de servicios.

Sección de Aseguradoras: Visualización de empresas asociadas mediante imágenes.

Formulario de Contacto: Envío de consultas mediante integración con Formspree.

Diseño Responsive: Adaptado a dispositivos móviles, tablets y escritorio.

Interactividad: Manejo de eventos y comportamiento dinámico con JavaScript.

Optimización para despliegue: Proyecto listo para producción en Netlify.

## 🛠 Tecnologías utilizadas

HTML5 – Estructura semántica del sitio web.

CSS3 – Diseño visual, estilos y responsive design.

JavaScript – Interactividad y manejo de eventos en el frontend.

PHP – Procesamiento básico del formulario en entorno local.

Formspree – Servicio externo para envío de formularios sin backend.

Netlify – Plataforma de despliegue y hosting del sitio.

## ▶️ Cómo ejecutar el proyecto

Clonar este repositorio en la carpeta htdocs (si usas XAMPP) o en el directorio raíz de tu servidor local.

Importar o verificar los archivos del proyecto dentro de la estructura correspondiente.

Ejecutar el servidor Apache (si trabajas en local con PHP).

Acceder al sistema desde el navegador web:
http://localhost/tu_carpeta/index.html

O bien acceder a la versión desplegada mediante Netlify.

## 🎯 Objetivo del proyecto

Desarrollar una web profesional para una empresa de seguros que permita:

Mejorar la presencia digital.

Facilitar el contacto con potenciales clientes.

Mostrar de forma clara las aseguradoras asociadas.

Contar con una solución rápida, escalable y fácil de mantener.
